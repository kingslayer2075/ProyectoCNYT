Calculadora de matrices y complejos
Introducción:
Esta calculadora fue desarrollada de forma manual siguiendo varios tutoriales que se pueden encontrar en varios canales de YouTube y Google por lo que están desarrolladas de una manera muy standard lo cual la hace de fácil entendimiento y sencilla manipulación.
para hacer unplugged del archivo se usó la aplicación de GitHub para computadora.

Requisitos: 
es necesario tener instalado y configurado Python y preferiblemente, aunque si bien opcional, un visualizador de código, en este caso fue usado Visual Studio Code.

Instrucciones:

Al ser un código de fácil uso, puede ser copiado directamente del repositorio de GitHub y pegado en algún editor de código cual quiera, debido a mi desconocimiento en el área no logre realizar una página de pruebas que pueda correr a la vez todas las funciones para hacer pruebas, por lo que lo realice de manera manual en un IDLE de Python, al ser pruebas muy ambiguas decidí no incluirlas en el repositorio.
